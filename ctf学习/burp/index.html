攻击模式介绍

Sniper：用于只爆破一个字段

Batter ram:用相同的值(字典)测试多个字段

例如： 尝试1尝试2.。。。

​ username aaa ttt

​ paseword aaa ttt

Pitchfork：

分别以不同的字典测试相应的字段

​ 使用场景：假设搞到了一个数据库，验证其用户名和密码能否匹配

例如： username aaa rrr

​ paseword 123 111
